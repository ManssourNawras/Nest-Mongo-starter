# Getting Started with NestJS Starter App

A simple Nest Starter App with Auth(Passport/JWT) and MongoDB

## using ...
- ### NestJs
- ### Typescript
- ### Passport/Jwt
- ### MongoDB

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
